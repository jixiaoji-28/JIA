# Jiaxian Ji — Research Portfolio

A no-build personal academic website designed for GitHub Pages.

## Preview locally

From this folder, run:

```powershell
python -m http.server 8000
```

Then open `http://localhost:8000`.

## Publish with GitHub Pages

1. Create a public GitHub repository named `jixiaoji-28.github.io`.
2. Copy the contents of this folder into the repository root.
3. Commit and push to the `main` branch.
4. In the repository, open **Settings → Pages**.
5. Under **Build and deployment**, choose **Deploy from a branch**, then select `main` and `/ (root)`.

The site will be available at:

`https://jixiaoji-28.github.io/`

## Main files

- `index.html` — all website content
- `system.css` — Swiss-grid layout, typography, colors, and responsive styles
- `script.js` — scroll progress and subtle section reveals
- `assets/tree-hero.png` — generated graphite tree cover artwork
- `assets/profile.jpg` — profile portrait
- `assets/Jiaxian_Ji_CV.pdf` — downloadable CV

Before publishing, review the research descriptions and replace the CV if a corrected version becomes available.
